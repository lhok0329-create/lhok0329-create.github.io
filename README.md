# Modern Personal Portfolio

A simple, modern, and responsive personal portfolio website template.

## Features

- **Modern Design**: Dark mode aesthetic with glassmorphism effects.
- **Responsive**: Adapts to different screen sizes.
- **Interactive**: Smooth scrolling and reveal animations.
- **No Build Step**: Just open `index.html` in your browser.

## How to Edit

1.  **Personal Details**: Edit `index.html` to change the name, navigation links, and content.
2.  **Styling**: Modify `style.css` to change colors, fonts, or layout.
3.  **Scripts**: Update `script.js` for additional interactivity.

## How to Run

Simply double-click `index.html` to open it in your web browser.
